Country Data Explorer

Version Control Practice with Git

In this project, I am practicing the use of Git for version control. Throughout this exercise, I create, update, and merge branches in Git. The project itself focuses on building a simple application that retrieves and displays country information using the REST Countries API. As I work on this project, I utilize technologies like HTML, CSS, JavaScript, and jQuery, while also learning how to manage my codebase effectively using Git.
